# AS.OPERATOR - Web Personal

Sitio web profesional de Angel Salinas especializado en automatizaciones con IA e integración de sistemas.

## 🚀 Características

- **Next.js 15** - Framework React moderno
- **Tailwind CSS 4** - Utilidades CSS avanzadas
- **Framer Motion** - Animaciones fluidas
- **TypeScript** - Tipado estático
- **Responsive Design** - Optimizado para todos los dispositivos
- **Calculadora Interactiva** - Cálculo de ahorros en tiempo real
- **Diseño Oscuro** - Tema ciber moderno

## 📋 Secciones

1. **Hero** - Presentación principal con llamada a la acción
2. **About Me** - Filosofía y especialidades
3. **Services** - Servicios principales (Automatización, Web, CRM)
4. **Process** - Metodología de trabajo (4 pasos)
5. **Calculator** - Calculadora interactiva de ahorros
6. **Footer** - Redes sociales y enlaces

## 🛠️ Instalación

```bash
# Clonar el repositorio
git clone https://github.com/angelsalinas79977-cmyk/WEB-PERSONAL-AS.git

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build
npm start
```
